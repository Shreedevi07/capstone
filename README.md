## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Shreedevi07/capstone.git
```

### 2. Navigate to the project

```bash
cd YOUR_REPOSITORY
```

### 3. Install backend dependencies

```bash
cd backend
npm install
```

### 4. Start the backend server

```bash
npm start
```

or

```bash
node server.js
```

(Replace `server.js` with `index.js` if that's your entry file.)

### 5. Run the frontend

Open the `frontend` folder in VS Code and start it using **Live Server**, or open `index.html` in your browser if your project doesn't require a development server.
